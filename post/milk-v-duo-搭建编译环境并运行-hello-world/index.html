Milk-V Duo 搭建编译环境并运行 Hello World

准备编译环境

  • Ubuntu 22.04 LTS

1.安装编译依赖的工具 

sudo apt-get install wget git make

2.将 duo-examples 仓库克隆到本地 

git clone https://github.com/milkv-duo/duo-examples.git

3.加载编译环境 

cd duo-examples
source envsetup.sh

下载完会自动解压到 duo-examples 下，解压后的目录名为 duo-sdk

4.编译测试 

cd hello-world
make

5.将编译好的程序 helloworld 传送到 Milk-V Duo

将编译成功后将生成的 helloworld 可执行程序通过RNDIS网络传送到 Milk-V Duo设备中

$ scp helloworld root@192.168.42.1:/root/

6.SSH Milk-V Duo

  • 远程链接Milk-V Duo (默认密码：milkv)
ssh root@192.168.42.1
  • 在ssh或者串口登陆的终端中运行编译好的程序，会打印 Hello, World!
[root@milkv]~# ./helloworld
Hello, World!
